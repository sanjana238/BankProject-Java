# Bank Management System - Java

A console-based banking application built using Core Java.

## Features
- Account creation with PIN authentication
- Deposit (min 1, max 10,000)
- Withdrawal (min 100)
- Balance enquiry
- PIN update with validation
- 3-attempt login security

## Concepts Used
- OOP (Encapsulation, Abstraction)
- Interface and Implementation
- 7 Custom Exception classes
- Factory Design Pattern
- Scanner for user input

## How to Run
1. Compile: javac Bank.java
2. Run: java Bank
